# 🎣 呉湾フィッシング・ナビゲーター Ultimate

広島県・呉湾エリアを対象にした
**釣りポイント可視化 × 環境分析 × AI釣果解析 × ダッシュボード** を統合した
Webベースの釣行サポートアプリです。

地図・天候・水温・釣果データを横断的に扱い、
「今日、どこで、何を狙うべきか」を直感的に判断できます。

---

## 🗺️ アプリ概要

-呉湾・周辺島嶼部の **25か所の釣りポイント** を収録
-天候・水温・魚種シーズンから **おすすめ度をスコア化**
-自然文で釣果を投稿 → **AIが自動解析してデータ化**
-日別・月別・年別の **釣果ダッシュボードを自動生成**

---

## ✨ 主な機能
**① 釣りポイント評価マップ（Leaflet）**

-OpenStreetMap + Leaflet によるインタラクティブマップ
-各ポイントに以下を表示：
　-総合スコア（0〜100）
　-現在の水温
　-今日の釣果数
　-評価理由（風・水温・シーズン・実績）
-**立入禁止・危険エリアは自動的に0点判定**

---

**② 環境データ分析**

-天候データ
　-風速をもとに「釣行可否」を判定
-水温データ
　-魚種ごとの適正水温と比較
-魚種シーズン
　-月単位でハイシーズン判定
　-シーズン外はスコア上限を制限（ロジック制御）

---

**③ 🐟 AI釣果チャット（OpenAI API）**

自然文で釣果を投稿するだけでOK。

例
```
阿賀マリノポリスでアジ3匹とメバル1匹釣れた
```

⬇️ AIが以下を自動抽出
-釣り場（表記揺れ対応）
-魚種（複数対応）
-匹数

⬇️ ダッシュボードに自動反映
-日別データ
-月間集計
-年間トレンド

※ 釣果チャットは 日付が変わると自動リセット

---

**④ 📊 釣果ダッシュボード（Chart.js）**

**■ 釣り場別・総釣果（棒グラフ）**

-当月の釣果を自動集計
-月が変わると自動リセット

**■ 釣り場別・月間トレンド（折れ線グラフ）**

-年間12か月分の推移を可視化
-「どの釣り場が、いつ強いか」が一目で分かる

---

**⑤ 🤖 AI釣り相談チャット**

-餌・仕掛け・狙い方をAIに相談可能
-釣果投稿とは独立した相談用AIチャット

---

**⑥ 🚻 周辺施設検索（OSM / Overpass API）**

-現在表示中の地図範囲から検索
-対応施設：
 -トイレ
 -駐車場
 -コンビニ
-マップ上にアイコンで表示

## 🗂️ データ設計（例）
**日別釣果データ**
```
dailyCatchData = {
  "阿賀マリノポリス": {
    total: 4,
    details: {
      "アジ": 3,
      "メバル": 1
    }
  }
}
```

**月間・年間データ**

**・**月間：棒グラフ用
**・**年間：月別折れ線グラフ用
**・**年・月が変わると自動初期化

## 🚀 起動方法
1. リポジトリをクローン
```
git clone https://github.com/yourname/kure-fishing-navigator.git
cd kure-fishing-navigator
```
2. ブラウザで起動
```
index.html を直接開くだけ
```

## 🔑 APIキー設定

画面右上の 「API設定パネル」 から以下を入力：

-呉市APIトークン
-海しる サブスクリプションキー
-OpenAI APIキー

※ APIキーは localStorageに保存
※ GitHubに公開する際はキーを含めないでください
